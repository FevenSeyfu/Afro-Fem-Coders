Introduction to CSS

CSS(Cascading Style Sheets) is the code that styles web content.

So what is CSS?

Like HTML,CSS is not a programming language.It's not a markup language either.CSS is a style sheet language.CSS is what you use to selectively style HTML elements.

Anatomy of a CSS element

Let's dissect the CSS code for red paragraph text to understand how it works:

The Individual parts in CSS ruleset are:

  1. Selector:This is HTML name at the start of the ruleset.It defines the element(s) to be styled.
  2. Declaration:This is a single rule like color:red;.It specifies which of the element's Properties you want to style.
  3. Properties:These are ways in which you can style an HTML element. (in this example,color is a property of the <p>elements.)
  4. Property value:This chooses one out of many possible appearances for a given property.(For example,there are many Color values in addition to red.)

Summary

CSS is a style sheet language used to selectively style HTML elements.The individual parts in ruleset covered in this lesson are..

  • Selector
  • Declaration
  • Properties
  • Property value
